
# Biblio pocket movile

Proyecto de uso personal para cargar libros (titulo, autor, cantidad de paginas y portada.). Con el mismo es posible poner una fecha de inicio de lectura, un estado (leyendo, por leer, terminado) y por ultima una valoracion del mismo una vez finalizado.





## Authors

- [@leonfelipecoll](https://www.github.com/felipecoll)


## Deployment

Proyecto creado con tecnologias como: 
* React
* Vite
* Tailwind
* NPM 
* Supabase
